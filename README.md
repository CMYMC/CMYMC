
#  안녕하세요


## 📂 Project

###  [**자세어때**](https://github.com/CMYMC/Project-AI-Smart-Gym)

RaspberryPi4 + Halio8(NPU)를 이용하여 사용자의 동작을 카메라 포즈 인식하여IMG·IMU 센서를 결합해, 스쿼트 등 운동 동작을 실시간으로 분류·채점하고 피로도·불균형까지 분석해주는 AI 스마트 피트니스 시스템

---
###  [**스마트팜**](https://github.com/CMYMC/Project-SmartFarm)
STM32, FreeRTOS를 기반으로 온습도 및 수위 데이터를 실시간 모니터링하고, 환경 변화에 따라 팬(Fan)과 펌프(Pump)를 자동 제어하여 최적의 환경을 조성하는 스마트팜 시스템 개발

---

###  [**스마트카**](https://github.com/CMYMC/Project-SmartCar)
STM32, Arduino, Raspberry Pi, MariaDB를 이용하여 전방 장애물 거리에 따라 차량 속도를 자동 제어, 속도 및 거리 등 운행 정보를 서버DB에 실시간 전송 및 저장하고 모니터링하는 안전 시스템 개발
