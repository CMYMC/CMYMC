##  Project

###  [**Relay-Bot(재난 통신 복구 로봇)**](https://github.com/CMYMC/Project-RelayBot)

화재나 지진 등 재난 상황에서는 통신 인프라(기지국, 공유기)가 붕괴되어 고립되는 경우 Relay-bot은 이러한 환경에서 사전 구축된 인프라 없이 스스로 통신 음영 지역을 찾아 이동하며 끊어진 네트워크를 복구하는 자율 주행 중계 로봇 시스템


###  [**Linux Device Driver Clock**](https://github.com/CMYMC/Device-Driver-Clock)

DS1302 RTC 모듈을 통해 정확한 시간을 유지하고, I2C OLED 디스플레이에 현재시간을 시각화하고 별도의 유저 공간 프로그램 없이 커널 드라이버 자체적으로 시간 설정 및 UI 제어 가능한 임베디드 시계 시스템 구축

---
###  [**자세어때(AI Smart GYM)**](https://github.com/CMYMC/Project-AI-Smart-Gym)

RaspberryPi4 + Halio8(NPU)를 이용하여 사용자의 동작을 카메라 포즈 인식하여IMG·IMU 센서를 결합해, 스쿼트 등 운동 동작을 실시간으로 분류·채점하고 피로도·불균형까지 분석해주는 AI 스마트 피트니스 시스템


---

###  [**대형차 전방 제어 시스템(SmartCar System)**](https://github.com/CMYMC/Project-SmartCar)
STM32, Arduino, Raspberry Pi, MariaDB를 이용하여 전방 장애물 거리에 따라 차량 속도를 자동 제어, 속도 및 거리 등 운행 정보를 서버DB에 실시간 전송 및 저장하고 모니터링하는 안전 시스템 개발

---
###  [**스마트팜**](https://github.com/CMYMC/Project-SmartFarm)
STM32, FreeRTOS를 기반으로 온습도 및 수위 데이터를 실시간 모니터링하고, 환경 변화에 따라 팬(Fan)과 펌프(Pump)를 자동 제어하여 최적의 환경을 조성하는 스마트팜 시스템 개발

---

###  [**STM32 Alarm Clock**](https://github.com/CMYMC/Project-Alarm-Clock) 
STM32 MCU의 내부 자원(Flash, Timer, ADC)을 최대로 활용한 디지털 알람 시계입니다. 단순한 시간 표시를 넘어, 상태 머신을 통한 체계적인 모드 관리와 Flash 메모리 제어를 통해 데이터 보존성을 구현 STM32F429ZI Nucleo 보드를 기반으로 제작한 알람 시계


### [**RTOS-CAN Gateway System**](https://github.com/Project-RTOS-CAN-Gateway-System) 
RT-CAN Gateway System은 여러 임베디드 보드에서 발생하는 실시간 센서 데이터를 CAN Bus로 수집하고 중앙 제어 보드에서 이를 통합·가공하고 데이터 흐름을 제어한 뒤 이더넷 네트워크로 전달 시스템
